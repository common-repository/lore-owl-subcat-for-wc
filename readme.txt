== Description ==

"Owl SubCat for WC" adds a simple Owl Carousel of WooCommerce product subcategories on top of the parent category pages, outside the product loop.

Just be shure to have set all the categories thumbnails, to have at least 1 product in every subcategory and to show only products in WooCommerce category pages (Customize > WooCommerce > Product Catalog > Category display).

Requires:
* WooCommerce 3.0 or newer (3.4+ recommended)
* PHP 5.4 or newer (7.2+ Recommended)
* WordPress 4.9 or newer

**Features**

* Add WooCommerce subcategories carousel ouside the product loop in category pages,
* Set number of items to display for every media device,
* Enable/Disable navigation arrows and dots,
* Set font size for subcategories title,
* Set space between items (padding),
* Set arrows position (top or bottom)

== Installation ==

1. Upload the plugin in the `/wp-content/plugins/` directory, or automatically install it through the 'New Plugin' menu in WordPress
2. Activate the plugin through the 'Plugins' menu in WordPress

= Plugin set-up =

1. Be shure to have set all the categories thumbnails;
2. Be shure to have at least 1 product in every subcategory;
3. In Customize > WooCommerce > Product Catalog > Category display, set the field to "Show products";
4. Make your choices in the plugin settings page and enjoy :)
